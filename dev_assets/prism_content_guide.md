## 가이드 1. 페르소나 명시

### 1.1 타겟 페르소나는 첫 5초 안에 식별 가능해야 한다

광고 첫 5초 안에 시청자가 "이게 나에게 하는 말이구나"를 인식해야 한다. 페르소나 식별이 모호한 광고는 ROAS 평균 2.1, 명확한 광고는 평균 4.3.

**구체화 방법**:
- 의상·환경·시간대·표정으로 직장인/엄마/학생 등 식별
- 자막 또는 발화로 명시 ("30대 직장인이라면", "워킹맘이라면")
- 이 둘 중 최소 하나는 5초 안에

### 1.2 페르소나-제품 매칭 정합성

페르소나가 명시되어도 제품과 미스매치면 의미 없음. 다음 매칭 가이드 참조:

| 카테고리 | 권장 페르소나 |
|---|---|
| 안티에이징 | 30대 후반~50대 여성 |
| 트러블 케어 | 10대 후반~20대 |
| 수분/기본 케어 | 전 연령 (단 한 영상 = 한 페르소나) |
| 콜라겐 등 이너 뷰티 | 30대~40대 여성 |
| 수면/스트레스 케어 | 20대~30대 직장인 |
| 키즈 영양제 | 30대~40대 엄마 |
| 프로틴/스포츠 | 20대~30대 운동인 |

## 가이드 2. 후크 · 첫 3초

### 2.1 첫 3초가 시청 완료의 60%를 결정

사내 분석: 첫 3초 시청률 80% 이상인 광고가 시청 완료율의 60%를 가져감. 첫 3초에 후크 없으면 광고는 끝.

### 2.2 권장 후크 패턴 (우선순위 순)

1. **사용자 1인칭 발화** — "저는 5년 동안…", "저요? 잡티 5년차였어요" — 사내 상위 콘텐츠 89% 채택
2. **상황 공감 질문** — "혹시 이런 증상 있으세요?", "오후 3시, 졸리지 않으세요?"
3. **즉각적 시각 임팩트** — Before 클로즈업 (단 사용자 발화와 함께일 때만)
4. **수치 충격** — "1,234명이 1주만에", "콜라겐 5,000mg"

### 2.3 사용자 검증 요소 (★ 핵심)

타율 상위 12개 콘텐츠 중 11개가 구체 사용자 숫자/후기 인용 포함. **이는 사내 가이드의 가장 강력한 패턴**.

**구체화 방법**:
- 사용 인원 수 명시 — "1,234명이 경험"
- 사용 기간 명시 — "5년차", "3개월차"
- 직접 화법 — "저는…", "제가…"

이 요소 누락은 검수 시 critical issue로 처리.

## 가이드 3. 시간·시점 명시

### 3.1 사용 시간 명시는 행동 유도력 강화

"퇴근 후 5분", "잠자기 전 1분", "출근길 30초" 같이 **언제·얼마나 시간이 드는지** 명시한 광고는 그렇지 않은 광고 대비 +14% engagement, +9% CVR.

### 3.2 효과 발현 시점도 구체적으로

"1주일이면", "30일 챌린지", "8주 임상 데이터" — **언제 결과를 볼 수 있는지** 명시는 신뢰도와 직접 연결.

### 3.3 직장인 타겟의 시간대 어필

평일 저녁 7~10시 / 주말 오후 시간대에 노출되는 광고는 "퇴근 후", "주말 아침" 같은 시간 컨텍스트를 활용 시 +18% CTR.

### 3.4 (요약) 시간 관련 카피의 3요소

- 사용 시간 ("5분")
- 사용 시점 ("퇴근 후")
- 효과 발현 시점 ("1주일이면")

이 3개 중 최소 2개 이상 포함을 권장.

## 가이드 4. 콘티 구조

### 4.1 표준 5-시퀀스 구조 (15초 영상 기준)

| 시퀀스 | 시간 | 목적 |
|---|---|---|
| 후크 | 0-3s | 페르소나 식별 + 시청자 잡기 |
| Before | 3-6s | 문제 상황의 시각적 증거 |
| 솔루션 | 6-10s | 제품 사용 루틴 (시간 명시) |
| After | 10-13s | 결과의 시각적 증거 |
| CTA | 13-15s | 행동 유도 + 한정 어필 |

이 구조에서 벗어나도 되지만, 각 5개 요소는 어떤 형태로든 포함되어야 함.

### 4.2 후크 임팩트 평가 기준

후크는 다음 5개 중 최소 2개 충족:
1. 페르소나 명시
2. 1인칭 발화 또는 상황 공감 질문
3. 시각적 임팩트 (얼굴 + 표정)
4. 자막으로 핵심 메시지 강조
5. 음악/사운드 임팩트 (모바일 환경 무음 시청 고려)

### 4.3 Before/After는 같은 인물·같은 환경

비교샷의 진정성은 두 화면이 같은 인물·같은 조명·같은 환경일 때 가장 강함. 다른 모델 사용 시 -22% trust score.

## 가이드 5. 시각 자료

### 5.1 Before/After는 사내 핵심 무기

뷰티 카테고리 ROAS 상위 콘텐츠의 78%가 Before/After 비교 포함. 각질·트러블·잡티·주름 등 시각적 변화가 명확한 카테고리는 필수.

### 5.2 자막 가독성 — 모바일 우선

90% 이상이 모바일·무음 시청. 자막은:
- 위치 — 화면 중앙~하단 1/3
- 크기 — 최소 화면 높이의 4%
- 색상 — 흰색 + 검정 외곽선 또는 검정 박스 위 흰색
- 길이 — 한 화면 최대 12자

### 5.3 클로즈업 가이드

- 인물 클로즈업: 얼굴 표정이 보이는 상반신
- 제품 클로즈업: 손에 들고 사용하는 모습 (제품만 단독 X)
- 부위 클로즈업: 변화가 보이는 부위 (잡티, 주름, 트러블 등)

## 가이드 6. CTA · 카피 톤

### 6.1 권유형 vs 명령형

30대 여성 타겟 대상 A/B 테스트: 권유형 CTA가 명령형 대비 +23% CVR.

| 명령형 (지양) | 권유형 (권장) |
|---|---|
| 지금 시작하세요 | 오늘 밤부터, 함께 시작해볼래요? |
| 클릭하세요 | 더 알아볼래요? |
| 구매하세요 | 한 번 써볼래요? |

단, 20대 남성 타겟에서는 명령형이 더 효과적 (+8%). 페르소나별 차이 고려.

### 6.2 한정 어필 + 카운트다운

CTA 영역에 다음 3개 중 최소 1개 포함:
- 한정 수량 ("선착순 100명")
- 한정 기간 ("오늘 자정까지", 카운트다운)
- 한정 가격 ("첫 구매 30%")

한정 어필 누락 시 평균 -7% CVR.

### 6.3 카피 길이

- 후크: 한 줄 (최대 15자)
- 메시지: 두 줄 (최대 30자)
- CTA: 한 줄 (최대 12자)
- 음성 보이스오버: 4초당 한 문장 이내

## 가이드 7. 채널·플랫폼 최적화

### 7.1 플랫폼별 길이

| 플랫폼 | 권장 길이 | 최대 |
|---|---|---|
| Instagram Reels | 12-15s | 30s |
| TikTok | 9-15s | 60s |
| YouTube Shorts | 15-20s | 60s |
| Meta Feed | 15-30s | 60s |
| Instagram Story | 5-15s | 15s (광고) |

### 7.2 플랫폼별 톤

- **Instagram**: 감성적, 라이프스타일, 미감 우선
- **TikTok**: 빠른 템포, 챌린지·트렌드 활용, 친근함
- **YouTube Shorts**: 정보 가치, 전문가성, 임상 근거
- **Meta Feed**: 안정적, 신뢰성, 장기 효과 어필

### 7.3 채널별 페르소나 분포

같은 페르소나라도 채널마다 다른 톤:

- 30대 여성 + Instagram → 감성·루틴 어필
- 30대 여성 + TikTok → 챌린지·즉각 효과 어필
- 30대 여성 + YouTube → 임상 근거·전문성 어필

## 가이드 8. 카테고리별 특수 가이드

### 8.1 건기식 — 안전성 어필

건기식 광고는 다음 요소 포함 권장:
- 식약처 인정 또는 임상 데이터 인용
- 부작용/주의사항 자막 (작게라도)
- "함께 복용 시 주의" 같은 책임감 있는 메시지

이 요소 포함 광고가 신뢰도 +31%.

### 8.2 트러블 케어 — 솔직 후기

10-20대 트러블 케어 카테고리는 솔직한 후기가 핵심:
- "5년 동안 트러블…" 같은 실제 사용자 시간
- 완전한 변화 X — 점진적 개선 강조
- "효과는 사람마다 다를 수 있어요" 솔직 어필 — 오히려 +14% trust

### 8.3 안티에이징 — 임상 데이터

40대+ 안티에이징은 시각적 후기 + 임상 데이터 둘 다 필요:
- 4주/8주 임상 결과 그래프
- 사용자 후기와 임상 데이터 병행

## 가이드 9. 정기 구독 어필

### 9.1 구독 어필은 4-6주 사용 후 광고에서

신규 고객 광고에서 정기 구독 어필은 -12% CVR (부담감). 첫 구매 권장.

리타게팅 광고 (4-6주 사용 후) 또는 정기성 강한 카테고리 (영양제, 데일리 케어)에서만 첫 광고에 구독 어필 가능.

### 9.2 구독 어필 시 강조 포인트

- 가격 절약 (30% 할인 등)
- 배송 편의 (자동 배송)
- 해지 자유 ("언제든 해지")

세 번째 "해지 자유"는 필수 — 누락 시 -19% 구독 전환.

## 가이드 10. 시즌·이벤트

### 10.1 시즌별 메시지 가이드

| 시즌 | 키워드 | 카테고리 |
|---|---|---|
| 봄 (3-5월) | 환절기, 리프레시, 톤업 | 수분 케어, 비타민 |
| 여름 (6-8월) | 더위, 자외선, 수분 보충 | 선케어, 수분, 마그네슘 |
| 가을 (9-11월) | 환절기, 트러블, 면역 | 트러블 케어, 면역 영양제 |
| 겨울 (12-2월) | 건조, 보습, 컨디션 | 보습 케어, 비타민 D, 면역 |

### 10.2 이벤트 타이밍

- 출근/등교 첫 주 (1월, 3월, 9월) — 컨디션·영양제
- 휴가 시즌 직전 (5-6월) — 다이어트·바디
- 연말 (11-12월) — 선물·세트
- 신년 (1월) — 다짐·시작 어필

## 가이드 11. 금기 표현

### 11.1 효능 과장 금지

식약처 가이드라인 준수:
- "치료" 표현 X (의약품만 가능)
- "100% 효과" 같은 단정 X
- "완치", "근본 해결" 같은 표현 X

대신:
- "도움을 줄 수 있습니다"
- "사용자의 90%가 만족"
- "임상 결과 OO 개선"

### 11.2 비교 광고 시 직접 브랜드명 X

경쟁사 직접 비교는 법적 리스크. 다음 표현 사용:
- "타사 일반 제품 대비"
- "시중 비슷한 가격대 제품 대비"
